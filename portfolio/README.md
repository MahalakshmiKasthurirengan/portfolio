 *Installation & Setup*
To run this project locally:

Clone the repository


```
git clone https://github.com/your-username/your-portfolio.git
cd your-portfolio
```
Install dependencies

```
npm install
Run the development server
```

```
npm run dev
Open http://localhost:5173/ in your browser.
```
